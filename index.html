My Skills

I work with the usual suspects—HTML, CSS, and the rest of the web toolbox. Sometimes I’m deep in code, other times I’m dragging and dropping in HubSpot. Either way, the goal’s the same: make things that actually work (and look good doing it).

HTML

I write clean, semantic HTML that search engines and screen readers actually like. No div soup, no markup crimes.

HTML5

I have a solid understanding of HTML5 and its role as the foundation of the web. From semantic markup and accessibility best practices to structuring pages for SEO and performance, I use HTML with intention and precision. Whether it’s building static websites or integrating with CMS systems like HubSpot, clean HTML is always at the core.

CSS3

Layouts, animations, weird edge cases—I wrangle CSS so it looks good and makes sense later. (Future me appreciates this.)

CSS3

CSS is where I bring designs to life. I work with everything from basic layout styling to advanced selectors, animations, transitions, and media queries. I have experience writing vanilla CSS from scratch, organizing styles logically, and using methodologies like BEM for clean, maintainable code.

Responsive

Built mobile-first, because people scroll more on their phones than their laptops. No awkward sideways swiping required.

Responsive

Creating experiences that look great on every device is second nature to me. I use fluid layouts, relative units, and flexible images—paired with mobile-first breakpoints—to ensure usability and visual integrity across screen sizes.

TailWind CSS

Utility classes keep things fast and consistent. Less “fighting with stylesheets,” more “hey, it just works.”

TailWind CSS

Tailwind is my go-to CSS framework for rapidly developing responsive, utility-first designs. I love how it streamlines development and eliminates the need to context-switch between HTML and CSS files. I use Tailwind to prototype fast and iterate even faster while keeping designs consistent across breakpoints.

Daisy UI

Prebuilt components on top of Tailwind. Great for saving time, then breaking them apart so the site doesn’t look like everyone else’s.

Daisy UI

DaisyUI makes styling even faster and more elegant on top of Tailwind CSS. I use DaisyUI’s pre-built, customizable component classes to speed up development without sacrificing flexibility. It helps me maintain consistent design patterns and create polished interfaces with less effort, especially when prototyping or scaling projects.

Bootstrap

Sometimes you just need the classics. I use Bootstrap when it fits, but not in a “hello, I’m clearly a Bootstrap site” kind of way.

Bootstrap

With Bootstrap, I can quickly spin up responsive layouts using its grid system, UI components, and JavaScript plugins. I’ve used it for building everything from MVPs to marketing pages and have customized it extensively to align with brand identities and design requirements.

HubSpot CMS

Pages, blogs, emails—you name it. I keep things modular and smart so marketing teams don’t curse my name later.

HubSpot CMS

I'm highly skilled in using HubSpot’s drag-and-drop editor to build professional, responsive pages and emails without needing to touch custom code. I work efficiently with modules, smart content, and form integrations to create marketing-ready assets that drive engagement. Whether it’s landing pages, blogs, or email campaigns, I know how to structure and optimize everything directly within HubSpot’s user-friendly tools.

WordPress

Themes, plugins, block editor. Set up clean so you can run it without sacrificing three hours every time you update a plugin.

WordPress

I have experience building and managing websites on WordPress, using its block editor (Gutenberg) and popular themes and plugins to create customized, responsive sites. I'm comfortable handling page layouts, basic plugin configurations, and making design adjustments to ensure a seamless user experience without heavy reliance on custom coding.
